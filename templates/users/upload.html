How to

You need to upload 2 files: *.xml and *.csv.

- CSV file must contain comma separated user data with header column names like: username, password, date_joined.

- XML file must contain next fields: first_name, last_name, avatar(url).

! All fields in that files are mandatory, otherwise user - will be ignored.

! Both files also must contain same users. User, that was only found in one file - will be ignored.
